# Reference: 紫微斗數 12 Palace + 14 Bintang Utama

Reference untuk interpretasi bagian Zi Wei Dou Shu di software 星僑.
Sistem: **三合派** (San He / Mainstream Taiwan school).

## Konsep Inti

紫微斗數 ("Astrologi Bintang Ungu") menggunakan **bintang virtual** yang
ditempatkan di papan **12 palace** berdasarkan jam-tanggal-bulan-tahun lahir
(kalender lunar). Berbeda dari BaZi yang fokus 干支, Zi Wei fokus ke posisi
bintang.

Software 星僑 V2.6 mengikuti sekolah **三合派** dengan elemen 四化.

### Aturan Per Palace (Quick Reference)

**命宮 (Palace Hidup)**
- Bintang utama → personality archetype
- Kosong (空宮) → pinjam dari 對宮 (遷移)
- Contoh: 紫微 di 命宮 = pemimpin alami; 七殺 = pejuang independen; 貪狼 = multitalented hedonis

**兄弟宮 (Palace Saudara)**
- 紫微/天府 = saudara sukses
- 七殺/破軍 = konflik dengan saudara
- 太陰廟 = saudari banyak/dekat

**夫妻宮 (Palace Pasangan)**
- 太陽 (untuk wanita) = suami terkemuka
- 太陰 (untuk pria) = istri lembut
- 七殺/破軍/貪狼 = pernikahan bergejolak
- 天同/天相 = harmonis

**子女宮 (Palace Anak)**
- 貪狼/廉貞 = anak banyak/aktif
- 七殺/破軍 = anak independen, sulit dekat
- 太陰 = anak perempuan dominan

**財帛宮 (Palace Rezeki)**
- 武曲/天府 = kaya stabil
- 貪狼+火/鈴 = wealth windfall (火貪格)
- 破軍 = boros, financial up-down
- 巨門 = uang dari mulut (sales/lawyer)

**疾厄宮 (Palace Kesehatan)**
- 廉貞 = darah/jantung
- 巨門 = mulut/pencernaan
- 太陰 = ginjal/hormon
- 武曲 = paru/pernafasan
- 火星 = inflamasi

**遷移宮 (Palace Perpindahan)**
- 紫微/天府 = bantuan dari atasan luar
- 七殺/破軍 = banyak pindah
- 天機 = sering bepergian

**僕役宮 (Palace Bawahan)**
- 紫微 tanpa 輔弼 = sendirian
- 貪狼 = teman banyak
- 巨門 = digosipkan teman

**官祿宮 (Palace Karir)**
- 武曲 = finance/military/teknik
- 廉貞 = administrasi/hukum
- 太陽 = publik figure
- 天機 = analis/agama
- 巨門 = guru/lawyer/sales

**田宅宮 (Palace Properti)**
- 天府/太陰 = properti banyak
- 破軍 = sering pindah rumah
- 火星/鈴星 = kebakaran/listrik (waspada)

**福德宮 (Palace Berkah)**
- 天同/天梁 = hidup nyaman
- 七殺/破軍 = gelisah
- 貪狼 = hedonis
- 巨門 化忌 = mood swings

**父母宮 (Palace Orangtua)**
- 太陽 廟 = ayah hebat
- 太陰 廟 = ibu hebat
- 化忌 = ortu sakit/jauh
- 廉貞+煞 = konflik dengan ortu

## 三方四正 (Triple-Square-Four-Right)

Untuk **setiap palace X**, "kekuatan total" = X + 對宮 + 三合 (dua palace
membentuk segitiga).

Rumus posisi:
- 本宮 = palace utama (posisi X)
- 對宮 = palace +6 (seberang papan)
- 三合 = palace +4 dan palace +8

Contoh: 命宮 di 子 → 三方四正 = 子 (本) + 午 (對) + 申 (三合) + 辰 (三合)

**Trinitas Penting:**

| Trinitas | Palace yang dibentuk | Makna |
|----------|---------------------|-------|
| **命財官** | 命 + 財帛 + 官祿 | Trinitas karir-uang (paling penting) |
| **夫官福** | 夫妻 + 官祿 + 福德 | Pasangan + karir + emosi |
| **子田疾** | 子女 + 田宅 + 疾厄 | Properti + anak + kesehatan |

**Aturan**: jika 本宮 kosong, pinjam bintang dari 對宮 (kekuatan 50%).

## Pola Kalimat Interpretasi Standar (Software 星僑)

Format kalimat standar:
```
「[宮位][主星][廟旺利陷]」
+ 主 [核心特質]，
+ 宜 [建議職業/行為]，
+ 忌 [警戒事項]。
+ 若見 [其他星/四化]，則 [修asi penjelasan]。
```

### Contoh per palace:

**命宮:**
- "命宮天機在子，廟旺，主聰明機智，心思細膩，善於策劃，惟思慮過多，性情多變"
  → "Palace Hidup berisi 天機 (Strategist) di posisi 子 (廟旺/penuh), bertanda
     cerdas, hati halus, jago menyusun rencana — namun kecenderungan
     overthinking dan mood berubah-ubah."

- "紫微在午，居廟，為帝王之尊，主貴氣顯赫"
  → "紫微 (Bintang Kaisar) di 午, di posisi 廟 (penuh) — bertanda kewibawaan
     menonjol, bakat kepemimpinan tinggi."

**夫妻宮:**
- "夫妻宮太陰廟旺，配偶溫柔賢淑，貌美多情"
  → "Palace Pasangan berisi 太陰 (Bulan) 廟旺 — pasangan lembut, anggun,
     dan penuh kasih sayang."

**財帛宮:**
- "財帛宮武曲化祿，財源廣進"
  → "Palace Rezeki berisi 武曲 (Eksekutor) yang ter-化祿 — sumber rezeki
     berlimpah."

**官祿宮:**
- "官祿宮太陽在巳午，居廟旺之地，主事業光明顯達"
  → "Palace Karir berisi 太陽 (Matahari) di 巳/午 (廟旺) — karir cemerlang
     dan menonjol."

## Catatan Tambahan

- **大限** = siklus 10-tahunan, mulai dari palace tertentu dan bergerak per
  10 tahun
- **小限** = siklus 1-tahunan, untuk fokus tahun tertentu
- **流年** = tahun berjalan (annual transit)
- **流月/流日** = bulan/hari berjalan

Software 星僑 menampilkan 大限 di tiap kotak palace dalam format `XX-YY` (misal
`23-32` artinya palace ini aktif untuk usia 23-32).
