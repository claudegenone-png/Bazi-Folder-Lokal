# Catalog 神煞 (Bintang Nasib) — 32 Bintang Utama

Reference untuk bagian 【神煞】 di output software 星僑.
Sumber: 三命通會, 子平真詮, plus tradisi Taiwan.

## Format Output Standar di Software

Setiap baris di software berformat:
```
【NAMA_HANZI】
· [karakteristik 1，karakteristik 2，karakteristik 3]
```

**Pola translate ke Indonesia:**
```
【Nama Hanzi · Pinyin · Arti Indonesia】
• [karakteristik 1, karakteristik 2, karakteristik 3]
```

Pola interpretasi standar (3-4 frase):
1. **Inti positif/peringatan** (frase pertama)
2. **Manifestasi perilaku** (frase kedua)
3. **Sisi sosial/positif** (frase ketiga)

## A. Bintang Mulia (貴人吉星)

### 1. 天乙貴人 (Tiān Yǐ Guì Rén) — *Bintang Pelindung Surgawi*

**Rumus** (口訣):
> 甲戊庚牛羊，乙己鼠猴鄉，丙丁豬雞位，壬癸兔蛇藏，六辛逢虎馬

Cara hitung: dari 日干 (atau 年干), lihat 地支 yang muncul.

**Indonesia**: "Bintang Pelindung" / "Mentor Star"
**Arti**: pertolongan saat krisis, mentor/atasan baik
**Kalimat standar**: "命帶天乙貴人，逢凶化吉，遇難呈祥，多得長輩貴人提攜"
→ "Memiliki Bintang Pelindung Surgawi — saat ancaman datang, berubah jadi
keberuntungan; sering dapat pertolongan dari orang yang lebih tua/atasan."

### 2. 太極貴人 (Tài Jí Guì Rén) — *Bintang Filosofi*

**Rumus**: 「甲乙生人子午中，丙丁雞兔定亨通，戊己兩干臨四季，庚辛寅亥祿豐隆，壬癸巳申偏喜美」

**Arti**: ketertarikan pada studi mistik/filsafat/agama
**Kalimat standar**: "好深思，研究玄學有成，文化底蘊深厚"
→ "Suka berpikir mendalam, tertarik filsafat/spiritual, latar budaya kaya."

### 3. 文昌貴人 (Wén Chāng Guì Rén) — *Bintang Akademik*

**Rumus**: 「甲乙巳午報君知，丙戊申宮丁己酉，庚亥辛子壬逢寅，癸人見卯入云梯」

**Arti**: kecerdasan, akademik, ujian, lulus dengan baik
**Kalimat standar**: "聰明過人，氣質儒雅，學業有成，逢凶化吉"
→ "Cerdas luar biasa, berkelas, sukses akademik, masalah berubah jadi berkah."

### 4. 學堂 / 5. 詞館 — *Akademi & Lexicon*

Rumus: 12 long sheng dari 日干. 學堂=長生, 詞館=冠帶
**Arti**: bakat akademik tinggi, cocok jadi guru/penulis/peneliti
**Kalimat**: "文采出眾，宜教學、寫作、研究"

### 6. 國印貴人 (Guó Yìn) — *Cap Negara*

## B. Bintang Asmara (桃花 / 紅艷)

### 15. 桃花/咸池 (Táo Huā / Xián Chí) — *Bunga Persik*

**Rumus** (dari 年支 atau 日支):
- 申子辰 → 酉
- 寅午戌 → 卯
- 巳酉丑 → 午
- 亥卯未 → 子

**Arti**: daya tarik, romance, sensualitas, sosialitas
**Kalimat standar (foto user contoh)**:
"色慾心太強，好色好酒，愛美，人緣好"

→ Translate Indonesia:
"Hasrat cinta sangat kuat, suka kemewahan dan alkohol, mencintai keindahan,
populer dalam pergaulan."

**Logika derivasi**:
- 桃花 ada di 子午卯酉 (4 cabang Cardinal/旺地) = energi sensual murni
- "色慾心太強" = 桃花 berlebih atau bertemu 七殺/傷官
- "好色好酒" = 子午卯酉 di tradisi terhubung sake culture
- "愛美" = 桃花 = aesthetics
- "人緣好" = magnetik, charm

**Variasi桃花 berdasar posisi**:
- 桃花 di 年支 = pre-marital romance, ortu romantis
- 桃花 di 月支 = orangtua/saudara romantis
- 桃花 di 日支 = pasangan attractive
- 桃花 di 時支 = anak attractive atau diri sendiri di akhir hidup

### 16. 紅艷煞 (Hóng Yàn Shà) — *Cantik Memikat*

**Rumus**: 「多情多慾少人知，六丙逢寅辛見雞，癸臨申上丁見未，甲乙午申庚見戌，戊己怕辰壬怕子」

**Arti**: lebih sensual/karismatik dari 桃花
**Kalimat**: "風流多情，異性緣極佳，但易招感情糾紛"
→ "Romantis, mudah menarik lawan jenis, tapi rentan kompleksi asmara."

## Posisi 神煞 di 4 Pillar

Posisi 神煞 di pillar mana mempengaruhi domain yang terpengaruh:

| Pillar | Domain | Usia | Hubungan |
|--------|--------|------|----------|
| 年柱 | leluhur, ortu jauh | 0-15 | nenek-kakek |
| 月柱 | ortu, saudara | 16-32 | orangtua kandung |
| 日柱 | diri, pasangan | 33-48 | self & spouse |
| 時柱 | anak, hari tua | 49+ | anak-cucu |

Contoh:
- 桃花 di 日柱 → daya tarik & pasangan attractive
- 桃花 di 時柱 → anak attractive atau diri sendiri di hari tua
- 將星 di 月柱 → orangtua punya otoritas
- 華蓋 di 日柱 → diri sendiri artistik & kesepian

## Daftar Tambahan 神煞 (jarang muncul tapi mungkin ada di foto)

- 從掃 / 衰掃 — bukan dari 三命通會 standar; kemungkinan dialek Taiwan
  untuk 衰星. Translate sebagai "Bintang Pelemahan" atau cek konteks.
- 大耗 (Dà Hào) — Penghamburan
- 小耗 (Xiǎo Hào) — Hamburan Kecil
- 披麻 (Pī Má) — Berkabung (jangan diterjemahkan langsung — pakai
  "tanda perubahan transisi")
- 弔客 (Diào Kè) — Tamu Belasungkawa (sama, gunakan reframing)
- 喪門 (Sàng Mén) — Pintu Duka (sama, reframing)

**Aturan etika**: jangan terjemahkan langsung 披麻/弔客/喪門 dengan kata
"kematian" atau "duka". Pakai reframing seperti "perubahan besar dalam
keluarga" atau "transisi spiritual".
